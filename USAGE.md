# User Guide

This guide explains how to use the main features of `nect-app`.

## Dashboard Overview

The dashboard gives you a quick snapshot of:

- daily calorie progress
- protein intake
- completed tasks and points
- exercise activity

## Task Tracker

### Add a task

1. Open the Tasks page.
2. Click **+ Add Task**.
3. Enter the task name and points.
4. Click **Save Task**.

### Mark complete

- Click the task status button to toggle between **Completed** and **Not Completed**.
- Completed tasks count toward the app score.

### Delete a task

- Click the trash icon to remove a task immediately.

### Automatic cleanup

- Tasks older than 30 days are removed automatically from the database.

## Exercise Planner

### Create a template plan

1. Go to the Exercises page.
2. Fill in the exercise plan details.
3. Save the plan.

### Track completion

- Use the exercise page buttons to mark today’s workout as complete.
- Exercise data appears on the dashboard.

## Food Tracker

### Log a meal

1. Open the Food page.
2. Choose an item from the food database.
3. Enter quantity and food details.
4. Save the entry.

### Nutrition tracking

- The app calculates calories and protein for each entry.
- Daily totals appear on the dashboard.

## Tips

- Add common foods to the database for faster logging.
- Use the dashboard to maintain daily nutrition and task habits.
- Refresh the browser after deployment to confirm live updates.
